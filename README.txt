The Flesch Reading Ease Test determines the easiness to read a certain passage. 
The passage must be in a .txt file and in the same folder as the .py file.

As defined by the test:
Readability = 206.835 - (1.015 * Average Words per Sentence) - (84.6 * Average Syllables per Word)

Score Table:
Greater than 90: Very Easy
80-89.99: Easy
70-79.99: Fairly Easy
60-69.99: Standard
50-59.99: Fairly Difficult
30-49.99: Difficult
Less than 30: Very Confusing
